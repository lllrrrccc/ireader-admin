> 版本：v1.0

---

## 1. 产品概述

### 1.1 背景

英语周报阅读器是一款面向 K12 学生的英语阅读学习工具，包含阅读器端（学生使用）、教师端（教师管理班级和任务）、家长端（家长查看学生数据）和管理后台（运营/教研管理）。

AI 管理模块是管理后台的核心功能之一，用于配置 AI 导读、AI 翻译、AI 讲解的提示词，管理知识库文档，以及查看 AI 交互记录。

### 1.2 目标用户

- **教研人员**：配置 AI 提示词，确保 AI 输出符合教学标准
- **运营人员**：查看 AI 使用记录，监控内容质量，发现异常内容并隐藏
- **管理员**：管理知识库文档，控制 AI 功能开关

### 1.3 核心价值

1. 灵活配置 AI 提示词，支持调用用户信息和知识库内容
2. 收集完整的 AI 交互记录，便于教研分析
3. 知识库作为 AI 回答的额外数据来源，提升回答质量

## 4. 各页面详细功能

### 4.1 AI导读管理

#### 4.1.1 一级页面：导读记录表

**页面顶部：** 标题「AI导读管理」+ 右上角「AI导读配置」按钮

**筛选条件：**
- 用户ID（文本搜索）
- 学生姓名（文本搜索）
- 书籍名称（文本搜索/下拉选择）
- 时间范围（日期范围选择）
- 状态（全部/成功/生成中/失败）
- 可见性（全部/可见/已隐藏）

**表格列：**
| 列名 | 说明 |
|------|------|
| ID | 记录唯一标识 |
| 用户ID | 学生唯一标识（S+7位数字） |
| 学生姓名 | 冗余展示 |
| 年级 | 学生触发时的年级 |
| 书籍名称 | 触发导读的书籍 |
| 导读内容摘要 | 截取前N字 |
| 获取时间 | YYYY-MM-DD HH:mm:ss |
| 耗时 | x.xs |
| 状态 | 成功/生成中/失败 |
| 可见性 | 可见/已隐藏 |
| 操作 | 查看全文 / 隐藏/显示 / 批量导出 |

**隐藏操作：**
- 发现不合适内容 → 点击「隐藏」→ 二次确认 → 内容对用户不可见
- 已隐藏记录可「取消隐藏」
- 隐藏后记录仍在列表，可见性列显示「已隐藏」标记

#### 4.1.2 二级页面：AI导读提示词配置

**页面顶部：** 「← 返回AI导读管理」按钮 + 标题「AI导读配置」

**配置区域：**

| 配置项 | 类型 | 说明 |
|--------|------|------|
| 系统角色设定 | 文本域 | 定义 AI 导读助手的身份、语气、风格 |
| 导读生成模板 | 文本域 | 提示词模板，含变量占位符 |
| 模型选择 | 下拉框 | GPT-4 / GPT-3.5 / Claude 3 |
| 保存 | 按钮 | 保存配置 |

**变量调用面板（右侧/下方）：**

```
┌─ 变量库 ──────────────────────────────┐
│ 分类：[用户信息 ▾] [知识库 ▾] [书籍 ▾]  │
│ ┌──────────────────────────────────┐  │
│ │ 🔍 搜索变量...                    │  │
│ ├──────────────────────────────────┤  │
│ │ ☐ {{昵称}}           [复制]      │  │
│ │ ☐ {{年级}}           [复制]      │  │
│ │ ☐ {{阅读级别}}       [复制]      │  │
│ │ ☐ {{设备SN}}         [复制]      │  │
│ │ ☐ {{书名}}           [复制]      │  │
│ │ ☐ {{作者}}           [复制]      │  │
│ │ ☐ {{章节标题}}       [复制]      │  │
│ │ ☐ {{当前段落译文}}   [复制]      │  │
│ │ ☐ {{本章译文}}       [复制]      │  │
│ │ ☐ {{知识库:作者简介}}[复制]      │  │
│ │ ☐ {{知识库:作品背景}}[复制]      │  │
│ └──────────────────────────────────┘  │
│              [复制选中变量]             │
└──────────────────────────────────────┘
```

**变量交互说明：**
- 先选「分类」（用户信息/知识库/书籍），筛选对应变量列表
- 支持搜索框输入关键词定位具体变量
- 点击「复制」将占位符复制到剪贴板，粘贴到配置文本域中
- 可勾选多个变量后点「复制选中变量」批量复制

### 4.3 AI讲解管理

#### 4.3.1 核心规则

**对话轮数限制：** 最长只允许进行 3 轮沟通

**数据模型：**
- 用户首次沟通触发 → 新增一条记录
- 该记录存储用户和 AI 持续对话的完整内容（最多3轮）
- 一条记录 = 一次完整的多轮对话

#### 4.3.2 一级页面：对话记录表

**页面顶部：** 标题「AI讲解管理」+ 右上角「AI讲解配置」按钮

**筛选条件：**
- 用户ID（文本搜索）
- 学生姓名（文本搜索）
- 书籍名称（文本搜索）
- 对话轮数（1/2/3）
- 时间范围（日期范围）
- 状态（全部/成功/生成中/失败）
- 可见性（全部/可见/已隐藏）

**表格列：**
| 列名 | 说明 |
|------|------|
| ID | 记录唯一标识 |
| 用户ID | 学生唯一标识 |
| 学生姓名 | 冗余展示 |
| 年级 | 学生触发时的年级 |
| 书籍名称 | 讲解发生的书籍 |
| 首轮问题 | 用户首次提出的问题（含系统默认前缀） |
| AI首轮回复摘要 | 首轮回复截取前N字 |
| 对话轮数 | 本轮讲解共几轮（1~3） |
| 获取时间 | 对话首次触发时间 |
| 状态 | 成功/生成中/失败 |
| 可见性 | 可见/已隐藏 |
| 操作 | **查看完整对话** / 隐藏/显示 / 批量导出 |

#### 4.3.3 对话详情弹窗（点击「查看完整对话」）

```
│                                               │

└───────────────────────────────────────────────┘
```

**弹窗规则：**
- 按时间线展示每轮对话（用户问 → AI答）
- 每轮标注轮次（第1轮/第2轮/第3轮）
- 标注用户是否收藏了该轮讲解（📌 已收藏）
- 如某轮回答引用了知识库，标注来源
- 管理员可在此弹窗中直接「隐藏此对话」

#### 4.3.4 二级页面：AI讲解提示词配置

| 配置项 | 类型 | 说明 |
|--------|------|------|
| 解释风格 | 下拉框 | 简洁明了/详细展开/适合儿童/学术深度 |

| 模型选择 | 下拉框 | GPT-4 / GPT-3.5 / Claude 3 |
| 保存 | 按钮 | 保存配置 |

**变量面板：** 结构同 AI导读，变量额外包含 `{{划选原文}}` `{{当前章节}}` `{{当前段落译文}}` `{{本章译文}}` `{{知识库:作者简介}}` `{{知识库:作品背景}}` 等

## 5. 跨端数据流

| 数据流 | 起点 | 管理端作用 | 终点 |
|--------|------|-----------|------|
| AI导读 | 阅读器端进入书籍 → 自动触发 | 配置提示词模板（注入用户信息+知识库）→ 查看记录 | 阅读器端展示导读弹窗 |
| AI翻译 | 阅读器端划取文字 → 点击翻译 | 配置提示词模板 → 查看记录 | 阅读器端气泡展示译文 |
| AI讲解 | 阅读器端划取文字 → 点击讲解 → 多轮对话 | 配置首轮提示词+多轮规则 → 查看完整对话记录 | 阅读器端对话页面 |
| 知识库 | 管理端上传文档 → 解析 | 管理解析状态，配置可调用标签 | AI 提示词配置时引用 |

## 6. 非功能性需求

- **性能**：记录表支持分页，每页默认 20 条，支持跳页
- **安全**：隐藏操作需二次确认，防止误操作
- **兼容性**：管理后台为 PC 端 Web 应用，适配 1280px 以上屏幕
- **数据保留**：记录隐藏后数据保留，仅对用户不可见
